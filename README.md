Coin Auction Hub
Coin Auction Hub is a virtual auction platform where users bid on rare coins and collectibles using virtual currency. New users receive 1000 free coins on signup to explore and participate risk-free.

Features--
Bid on auctions with virtual coins, no real money involved

List your own items to earn coins from winning bids

Real-time auction updates with countdown timers and bid tracking

Users can purchase additional coins to keep bidding

Simple registration with instant coin credit

My Items section to track winnings and listings

Technical Stack--
Frontend: HTML5, CSS3, Vanilla JavaScript

Backend: Node.js and Express.js

Data Storage: MongoDB

Responsive and interactive UI with smooth animations

User Flow--
Register and get 1000 free coins

Browse live auctions and place bids

List items for auction with custom duration and starting bids

Win auctions and collect items

Purchase more coins as needed

Economy Model--
Coins deducted immediately when bidding (makes bidding strategic)

Sellers earn coins from winning bids

Real money converts to virtual coins through coin packages

Setup & Run
Clone repo

Install dependencies with npm install

Run server with node server.js

Open index.html in browser

Enjoy risk-free, gamified coin auctions and collectible trading!
